# lyricli
search song lyrics from the terminal

### usage
`$ lyricli <artist> <song>`  
![2023-11-01-203759_549x686_scrot](https://github.com/yazoink/lyricli/assets/98802603/958c3887-edb1-4565-8d3a-87ef5394eb99)

### installation
`$ git clone https://github.com/yazoink/lyricli`  
`$ cd lyricli`  
`$ ./install.sh`
