# lyricli
search song lyrics from the terminal

### usage
`$ lyricli <artist> <song>`  

![2023-11-01-204143_507x760_scrot](https://github.com/yazoink/lyricli/assets/98802603/9f9bdddd-4985-48e4-b9ce-220491ccd2f6)

### dependencies
- ![hq](https://github.com/coderobe/hq)
- curl

### installation
`$ git clone https://github.com/yazoink/lyricli`  
`$ cd lyricli`  
`$ ./install.sh`
