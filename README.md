# lyricli
search song lyrics from the terminal

### usage
`$ lyricli <artist> <song>`  

![2023-11-01-204143_507x760_scrot](https://github.com/yazoink/lyricli/assets/98802603/bc7cf24c-7cb9-44c7-91ce-6243c30b50fc)

### installation
`$ git clone https://github.com/yazoink/lyricli`  
`$ cd lyricli`  
`$ ./install.sh`
